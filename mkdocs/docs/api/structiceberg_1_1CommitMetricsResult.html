Detailed Description

Immutable snapshot of commit metrics for use in CommitReport.

Populated by CommitMetrics::ToResult() after a commit completes. File and record counts can also be combined from the snapshot summary with From().

Build a CommitMetricsResult from live metrics and a snapshot summary map.

Combines timer/retry measurements from live_metrics with records parsed from snapshot_summary. Missing or unparseable summary keys are omitted.
