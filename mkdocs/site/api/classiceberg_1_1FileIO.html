Detailed Description

Pluggable module for reading, writing, and deleting files.

This module only handle metadata files, not data files. The metadata files are typically small and are used to store schema, partition information, and other metadata about the table.

Note that these functions are not atomic. For example, if a write fails, the file may be partially written. Implementations should be careful to avoid corrupting metadata files.
